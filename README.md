## 🌍 Overview

A premium, rugged, mobile-first landing page for **NomaTrack** — the AI-powered platform that eliminates the 60% middleman tax by connecting smallholder farmers directly to buyers through orbital telemetry and the Harvest Score™ ecosystem.

Designed to feel like a serious venture-backed agro-tech company (think Anduril meets Planet Labs). Built as a single, high-performance HTML file with zero dependencies beyond Tailwind CDN and Font Awesome.

---

## ✨ Features

- Fully responsive & mobile-optimized (icons no longer cut off)
- Live scrolling satellite ticker
- Rugged "soil-tech" visual language with custom Bricolage Grotesque + JetBrains Mono typography
- Smooth reveal animations on scroll
- Working mobile hamburger menu
- Founder spotlight section with direct contact links
- Orbital Intelligence & Zero-Waste Logistics messaging
- Economic impact visualization
- Dark, high-contrast agro-tech aesthetic
